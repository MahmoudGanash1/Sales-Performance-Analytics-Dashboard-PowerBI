 📊 Sales Dashboard (Power BI)

 📌 Overview
This project is an interactive sales dashboard built using Power BI to analyze sales performance, profit trends, and customer behavior. The dashboard provides a clear and dynamic view of key business metrics.

 🛠 Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

 📈 Key Features
- Sales and profit tracking across different time periods
- Regional performance analysis
- Customer segmentation overview
- Salesperson performance tracking
- Monthly and daily sales trends
- Interactive filters (Year, Quarter, Ship Mode)

📊 Visualizations
- Bar charts for regional and monthly analysis
- Line chart for daily sales trends
- Donut charts for ship mode and customer segmentation
- KPI cards for top product and sub-category performance
